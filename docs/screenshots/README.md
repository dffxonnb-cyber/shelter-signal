# Shelter Signal Screenshots

이 폴더는 포트폴리오와 배포 문서에 사용할 앱 화면 캡처를 보관하기 위한 자리입니다.

권장 캡처 환경은 모바일 기준 `390x844`입니다. 필요하면 태블릿 `768x1024`, 데스크톱 `1280x900` 보조 캡처를 추가합니다.

## Capture Checklist

- `home`: 오늘의 보호 종료 신호, 우선 확인 공고, 지역 신호 요약이 보이는 홈 화면
- `golden-time`: 보호 종료가 가까운 공고 중심의 골든타임 화면
- `notice-list-filters`: 공고 필터와 표시 수 조절 컨트롤이 함께 보이는 화면
- `detail-sheet`: 공식 공고 정보, 동물 정보, 보호소 및 연락처, 문의 안내 액션이 보이는 상세 시트
- `region-explorer`: 권역 선택 패널, 개별 지역 칩, 선택한 지역 summary가 보이는 지역 탐색 화면
- `saved-placeholder`: 관심 공고 저장 placeholder 화면
